# 🍕 Food Ordering App

A full-stack food ordering web application built with React and Django.

## Team Members
- Member 1: Anique Bhatti
- Member 2: Mohsin Raza

## Tech Stack
- Frontend: React JS, CSS3
- Backend: Django, Django REST Framework
- Database: SQLite
- Authentication: JWT

## Features
- User Registration & Email Verification
- Login/Logout
- Browse Menu with Search & Filter
- Add to Cart
- Place Orders
- Order History & Tracking
- Admin Dashboard with Charts
- Food Item Management (CRUD)
- Image Upload for Food Items
- Real-time Notifications

## Setup Instructions

### Backend
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

### Frontend
cd frontend
npm install
npm start
